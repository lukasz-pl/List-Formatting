# Advent Calendar

## Podsumowanie
This sample demonstrates an advent calendar where you can click on the present box of the day to see the presents. The present box cannot be opened if the value in the `Data` column is later than today.

![zrzut ekranu próbki](./assets/screenshot.gif)

## Wymagania widoku

|Type               |Internal Name|Wymagane|
|-------------------|-------------|:------:|
|Pojedyncza linia tekstu|Title        |Yes     |
|Thumbnail          |Present      |Yes     |
|Data i godzina      |Data         |Yes     |
|Yes/No             |Opened       |Yes     |
|Managed Metadata   |BoxColor     |Yes     |
|Managed Metadata   |RibbonColor  |Yes     |

- You need to use the Gallery View.
- `BoxColor` and `RibbonColor` should be set to the HTML color code or color name.
