# Multilingual Quotes

## Podsumowanie

This sample demonstrates the use of the `@UIlcid` token to change the text displayed based on the site display language.

![zrzut ekranu próbki](./assets/screenshot.png)

## Wymagania widoku

|Type                |Internal Name|Wymagane|
|--------------------|-------------|:------:|
|Pojedyncza linia tekstu |Title        |Yes     |
|Pojedyncza linia tekstu |Name         |Yes     |
|Pojedyncza linia tekstu |Japanese     |Yes     |
|Pojedyncza linia tekstu |JapaneseName |Yes     |
