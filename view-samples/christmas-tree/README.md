# Christmas Tree

## Podsumowanie
This sample draws a tree across rows in 3 different stages "**alt**" as top, "**body**" as tree and "**wood**" as bottom. You can also include icons in the tree. The **Title** field will appear as a tooltip on the icons.

## Wymagania widoku
- The format expect the following fields:

Field |Type
--------|---------
Title | Pojedyncza linia tekstu 
Structure | Pojedyncza linia tekstu - The tree is composed by the following structure: "**alt**" - Header of Tree (**Triangle**), "**body**" - body of tree (**Trapezoid**) and "**wood**" - Base of Tree (**Smaller Trapezoid**)
color | Pojedyncza linia tekstu - (recommended colors for the tree **79a83f**, **7cad40**, **c19c00**. For the base of tree recommend color **7e5224**). PS: no need to include #
Gift |  Pojedyncza linia tekstu - icons of gifts (samples **GiftboxOpen**, **GiftCard**, **Giftbox**, **GiftBoxSolid** and **FavoriteStarFill** for top of tree, )
GiftSide | Pojedyncza linia tekstu - options "**Left**" or "**Right**"

### Edit View requirements

- Edit View where format will be included, access to "**Sort**" Area and select column "**Structure**" and check as **ascending order**, this option order tree correctly from top to bottom event when included new levels in the tree.
