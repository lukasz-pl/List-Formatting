# Location Bing Maps

## Podsumowanie
Ta próbka pokazuje how to use a Location column to display details and a Map from Bing.

![zrzut ekranu próbki](./assets/screenshot.gif)

## Wymagania widoku
- Ten format można zastosować do any `Location` Column.
