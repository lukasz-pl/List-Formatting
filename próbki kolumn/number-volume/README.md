# Volume Option

## Podsumowanie
Ta próbka zawiera visual selection of volume from 0 to 5.

![zrzut ekranu próbki](./assets/screenshot.gif)

## Wymagania widoku
- Format oczekuje następujących pól:

Pole |Typ
--------|---------
Volume | Liczba - Volume options from 0 to 5
