# Volume Option

## Podsumowanie
Ta próbka zawiera visual selection of volume from 0 to 5.

![zrzut ekranu próbki](./assets/screenshot.gif)

## Wymagania widoku
- Format expect the following fields:

Field |Type
--------|---------
Volume | Number - Volume options from 0 to 5
